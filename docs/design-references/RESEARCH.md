# Design Research Notes

Mobbin MCP unavailable. Research synthesized from known swipe-deck patterns and Apple glass UI conventions.

## Swipe deck patterns (Tinder / Bumble / Hinge)

- Single focal card with peek of next card behind
- Horizontal drag with rotation proportional to offset
- Explicit pass/like buttons below card for users who don't swipe
- Commit threshold ~25-30% of card width

## Glass UI patterns (Apple Liquid Glass)

- Semi-transparent surfaces with backdrop blur
- Thin luminous border (1px white at ~25% opacity)
- Content on glass uses opaque text backing where needed for readability
- Large corner radius (20-28px) on cards and sheets

## Applied to Relationship Memory

- Dark gradient base (not white dating-app aesthetic)
- Keep/Forget instead of Like/Pass — calmer accent colors
- Event Name always visible — this is memory, not romance
